- ### **编译**
  - **`如果你需要Clone该项目，在Clone完成后请进入到该项目的根目录下并输入以下命令才可正常编译`**
  - **`rmdir lib_mangax/src/main/jni/OpenCC`**
  - **`git init`**
  - **`git submodule add -f --name OpenCC -b master https://github.com/BYVoid/OpenCC.git lib_mangax/src/main/jni/OpenCC`**
